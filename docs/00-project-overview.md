# Project Overview

## Vision

Finance Goblin is a small ESP32-based physical AI companion designed to live on a developer's desk.

It continuously analyzes personal context, starting with expense data, and reacts using:

- LCD text
- RGB mood lighting
- buzzer sounds
- button interactions
- AI-generated personality commentary

The goal is not to create another dashboard. The goal is to create a tiny physical AI creature with personality and ambient awareness.

## Product Names

Possible names:

- Finance Goblin
- DeskPet AI
- Tiny CFO
- Chaos Totem
- Ambient Goblin
- Desk Creature

## Product Goals

Primary goals:

- Learn IoT, AI, and backend integration.
- Create a useful physical desk gadget.
- Build emotional attachment through personality.
- Practice embedded systems architecture.
- Create a long-term expandable side project.

Secondary goals:

- Build a strong portfolio project.
- Experiment with ambient AI concepts.
- Explore human-device interaction.
- Practice end-to-end system design.

## Product Philosophy

Optimize for:

- personality
- physical presence
- fun
- iteration speed
- emotional feedback
- usefulness

Avoid optimizing early for:

- large-scale architecture
- microservices
- Kubernetes
- distributed systems
- enterprise complexity
